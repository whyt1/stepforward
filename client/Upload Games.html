Submission Guidelines

Educational Value

All games must have clear educational objectives and align with standard learning outcomes for the target age group.

Content Appropriateness

Content must be age-appropriate with no violence, inappropriate language, or unsafe scenarios.

Performance Requirements

Game should run smoothly on standard devices with limited loading times and no crashes.

Data Privacy

Games must comply with all data privacy regulations and clearly disclose any data collected or stored.
